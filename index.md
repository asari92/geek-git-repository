# Руководство по работе с GIT

## Инициализация нового репозитория

Для инициализации нового репозитория введите блок кода:
```
    git init
```

## Добавление файла в репозиторий

Для добавления файла в репозиторий введите команду:
```
    git add <имя файла>
```

## Отслеживание состояния репозитория

Для отслеживание состояния репозитория введите команду:
```
    git status
```

## Вычисление разницы между любыми двумя Git деревьями

Для вычисления разницы между любыми двумя Git деревьями введите команду:
```
    git diff
```
